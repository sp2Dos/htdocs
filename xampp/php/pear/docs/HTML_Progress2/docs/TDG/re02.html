HTML_Progress2::hasErrors
Prev  Error Handler Next

Name

HTML_Progress2::hasErrors — Determine whether there are errors into the HTML_Progress2 stack

Synopsis

      require_once 'HTML/Progress2.php';

integer HTML_Progress2::hasErrors();
;

Description

This function gave you ability to be aware of API errors presence in user scripts.

Throws

throws no exceptions thrown

See

see HTML_Progress2::getError, HTML_Progress2::raiseError

Note

since 2.0.0

This function can not be called statically.
