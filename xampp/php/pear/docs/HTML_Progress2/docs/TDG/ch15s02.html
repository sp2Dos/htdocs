Quick-Cards Atttributes Reference

Note

Position/size are relative to progress bar itself (left), (top), (width), (height). That means the default text label is on top side of the progress bar.

Width or height sets to zero means there is no size restriction (== auto-size).

top <integer> 25
width <integer> 300
height <integer> 25

min <integer> 0
