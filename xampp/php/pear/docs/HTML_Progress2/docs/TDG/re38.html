HTML_Progress2::getProgressAttributes
Prev  Background color and size Next

Name

HTML_Progress2::getProgressAttributes — Returns the progress background attributes values

Synopsis

      require_once 'HTML/Progress2.php';
mixed HTML_Progress2::getProgressAttributes( $asString = FALSE);  
bool   $asString = FALSE;

Description

Get all background attributes in a associative array with key-value couple (by default). Get the attributes as string is also possible.

Parameter

boolean $asString

(optional) whether to return the attributes as string

See

see HTML_Progress2::setProgressAttributes

Throws

throws HTML_PROGRESS2_ERROR_INVALID_INPUT

Note

since 2.0.0

This function can not be called statically.
