Events Reference

onSubmit

This event is triggered by run() method when progress meter start process (0%)

onLoad

This event is triggered by run() method when progress meter end process (100%)

onChange

This event is triggered by setMinimum(), setMaximum(), setValue(), incValue(), moveStep(), and moveNext() methods each time the current value change.

onCancel

This event is triggered only by a progress monitor, when user decide (click on button) to stop process.
