HTML_Progress::isIndeterminate
determines whether the progress meter is in determinate mode or not

Synopsis

require_once 'HTML/Progress.php';

bool HTML_Progress::isIndeterminate()

Description

This method is used to know whether the progress meter is switched on determinate mode (default) or not (indeterminate).

Note

( HTML_Progress >= 1.0 )

This function can not be called statically.

See

HTML_Progress::setIndeterminate

Example

Look and feel of indeterminate mode used with HTML_Progress_Monitor.
