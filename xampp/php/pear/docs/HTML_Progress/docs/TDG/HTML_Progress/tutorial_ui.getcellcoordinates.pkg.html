HTML_Progress_UI::getCellCoordinates
returns the cell coordinates for a polygonal shape

Synopsis

require_once 'HTML/Progress/UI.php';

mixed HTML_Progress_UI::getCellCoordinates()

Description

This method is used to retrieve coordinates of progress meter cells (polygonal shape only).

Note

( HTML_Progress >= 1.2.0RC1 )

This function can not be called statically.

See

HTML_Progress_UI::setCellCoordinates

Example

This example will print the list of cell coordinates for a square 3x3 progress meter, defined as below:

Cell# (X,Y)
0 0,0
1 0,1
2 0,2
3 1,2
4 2,2
5 2,1
6 2,0
7 1,0
