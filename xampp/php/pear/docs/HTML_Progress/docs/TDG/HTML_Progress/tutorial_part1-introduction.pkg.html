Introduction
a guide to PEAR::HTML_Progress written for developers by a developer

Purpose of this text

Intention of this guide is to provide developers, with all necessary information to start working effectively.

The first part of this guide will cover all the features of this package by series of tutorial. After that we will have a full reference API guide. Finally the guide will point out, usage of mandatory and optional components to install, some advices to adjust your code from the version 1.1, and the most common usage problems into a FAQ.
