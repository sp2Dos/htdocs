Appendix A. Installation
how to obtain and install HTML_Progress

How to obtain HTML_Progress

You can download the latest version of HTML_Progress from its PEAR homepage http://pear.php.net/package/HTML_Progress/.

Installing

  1. Setup the PEAR package manager (if you haven't got it already) - see Getting the manager at http://pear.php.net/manual/en/installation.getting.php (command line version) - or Getting Up and Running with the PEAR Web Installer at http://www.phpkitchen.com/article.php?story=20021201062516970 (web package manager).

  2. Type pear install HTML_Progress (using the command line manager)

  3. PEAR::HTML_Progress is now installed

Requirements

In order to successfully use all features of HTML_Progress, you need also some others PEAR packages :

  • HTML_Common 1.2 or greater, is mandatory.
  • PEAR 1.3.1 or greater, is used only by the default error handler system to raise a PEAR_Error object.
  • HTML_QuickForm 3.1.1 or greater, is used by HTML_Progress_Monitor and HTML_Progress_Generator classes.
  • HTML_QuickForm_Controller 0.9.3 or greater, is used only by the HTML_Progress_Generator class.
  • HTML_CSS 0.3.4 or greater, is used by HTML_Progress class to generate default stylesheet.
  • HTML_Page 2.0.0RC2 or greater, is used only by the HTML_Progress_Generator class.
  • Config 1.9 or greater, is used only by the HTML_Progress::setModel method.
  • PHPUnit 0.6.2 or greater, is used only to run regression tests.
  • Image_Color 1.0.0 or greater, and PHP extension GD 2.0.1 or greater, are used only to produce circle or ellipse progress shape.
