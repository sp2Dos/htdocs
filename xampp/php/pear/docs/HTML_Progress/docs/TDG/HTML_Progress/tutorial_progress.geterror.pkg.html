HTML_Progress::getError
returns count of API error

Synopsis

require_once 'HTML/Progress.php';

mixed HTML_Progress::getError()

Description

This method pop an error off of the HTML_Progress stack. This item will be purged from stack.

Note

( HTML_Progress >= 1.2.0RC3 )

This function can be called statically.

The HTML_Progress error stack follows the same PEAR_ErrorStack structure: associative array with keys 'code', 'params', 'package', 'level', 'time', 'context'.

See

HTML_Progress::raiseError, HTML_Progress::hasErrors
